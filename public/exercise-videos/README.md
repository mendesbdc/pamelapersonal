# Vídeos dos exercícios

Coloque nesta pasta os vídeos ou GIFs próprios dos exercícios.

A tela do aluno procura automaticamente pelo mesmo nome do exercício:

- `Cadeira extensora.mp4`
- `Cadeira extensora.webm`
- `Cadeira extensora.gif`

Também funciona com o nome simplificado sem acento e com hífen:

- `cadeira-extensora.mp4`
- `cadeira-extensora.webm`
- `cadeira-extensora.gif`

Ordem de busca:

1. `.mp4`
2. `.webm`
3. `.gif`

Exemplos:

- Exercício `Agachamento` -> `Agachamento.mp4` ou `agachamento.mp4`
- Exercício `Elevação pélvica` -> `Elevação pélvica.mp4` ou `elevacao-pelvica.mp4`
- Exercício `Tríceps corda` -> `Tríceps corda.mp4` ou `triceps-corda.mp4`
